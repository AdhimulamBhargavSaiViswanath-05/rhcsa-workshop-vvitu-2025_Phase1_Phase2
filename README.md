## ğŸ¯ RHCSA (EX200) Exam Objectives

The RHCSA certification validates skills in the following areas:

### 1.  Understand and Use Essential Tools
- Access a shell prompt and issue commands
- Use input-output redirection
- Use grep and regular expressions
- Access remote systems using SSH
- Manage files and directories
- Create and edit text files

### 2. Operate Running Systems
- Boot, reboot, and shut down systems
- Boot systems into different targets
- Interrupt the boot process
- Identify CPU/memory intensive processes
- Adjust process scheduling
- Manage tuning profiles
- Locate and interpret system log files

### 3. Configure Local Storage
- List, create, delete partitions
- Create and remove physical volumes
- Assign physical volumes to volume groups
- Create and delete logical volumes
- Configure systems to mount file systems

### 4. Create and Configure File Systems
- Create, mount, unmount file systems
- Mount and unmount network file systems using NFS
- Configure autofs
- Extend existing logical volumes
- Create and configure set-GID directories
- Diagnose and correct file permission problems

### 5. Deploy, Configure, and Maintain Systems
- Schedule tasks using cron and at
- Start and stop services
- Configure systems to boot into specific targets
- Configure time service clients
- Install and update software packages
- Modify the system bootloader

### 6. Manage Users and Groups
- Create, delete, and modify local user accounts
- Change passwords and adjust password aging
- Create, delete, and modify local groups
- Configure superuser access

### 7. Manage Security
- Configure firewall settings
- Manage default file permissions
- Configure key-based authentication for SSH
- Set enforcing and permissive modes for SELinux
- List and identify SELinux file and process context
- Restore default file contexts
- Manage SELinux port labels
- Use boolean settings to modify system SELinux settings
- Diagnose and address routine SELinux policy violations
